# /convert — Конвертация валют

Конвертирует сумму из одной валюты в другую через официальный курс ЦБ РФ.

## Используемые инструменты

| Инструмент | Что делает |
|------------|------------|
| `convert_currency` | Конвертация через курс ЦБ |

## Алгоритм

1. Разбери запрос пользователя: сумма, исходная валюта, целевая валюта
2. Если целевая не указана — по умолчанию RUB
3. Вызови `convert_currency` с параметрами
4. Покажи результат с использованными курсами

## Формат ответа

```
## Конвертация 1000 USD → EUR

**Результат**: 923.45 EUR
**Курс USD**: 92.57 ₽
**Курс EUR**: 100.25 ₽
**Дата курсов**: 2026-03-30 (ЦБ РФ)
```

## Примеры

```
/convert 1000 USD в EUR
/convert 5000 EUR в рубли
/convert 100000 RUB в CNY
```
