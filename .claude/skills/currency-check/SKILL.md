# /currency-check — Проверка курса валюты

Показывает текущий курс валюты ЦБ РФ с изменением за день.

## Используемые инструменты

| Инструмент | Что делает |
|------------|------------|
| `get_currency_rate` | Курс конкретной валюты с изменением |
| `get_daily_rates` | Все курсы для сравнения |

## Алгоритм

1. Получи код валюты от пользователя (USD, EUR, CNY и т.д.)
2. Вызови `get_currency_rate` с этим кодом
3. Покажи курс, изменение за день, процент изменения
4. Если пользователь спросил несколько валют — вызови для каждой

## Формат ответа

```
## Курс USD/RUB (ЦБ РФ)

**Курс**: 92.5714 ₽ за 1 USD
**Изменение**: +0.3214 (+0.35%)
**Дата**: 2026-03-30
```

## Примеры

```
/currency-check USD
/currency-check EUR
/currency-check CNY
```
